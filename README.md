# Projet Speed Dating

## Description du Projet

Tinder est une application mondiale de rencontre en ligne.
L'équipe marketing de Tinder fait face à une diminution du nombre de matchs.
Elle cherche à comprendre les mécanismes qui suscitent l'intérêt mutuel entre deux personnes.

## Objectifs

* Statistiques desccriptives
* Graphiques
* Interprétation

## Données

Speed+Dating+Data.csv: Dataset d'événements de speed dating organisés entre 2002 et 2004
8 378 lignes, 195 colonnes

## Structure du projet

* ```data```: Jeux de données
* ```images/```: Images utilisées dans les notebooks
* ```notebook/```: Notebooks Jypiter
